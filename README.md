### Quick start

First install CMake, Qt5, OpenCV.

Then build and run.

```sh
git clone https://github.com/lenna-project/lenna
cd lenna
mkdir build
cd build
cmake ..
make
./bin/lenna
```

#### SNAP Package
```sh
make snap
```
